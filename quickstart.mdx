# Quickstart

This guide walks you through integrating Signum authentication into your application.

## Prerequisites

- Node.js 18+ or modern browser
- A Signum client ID (or use development mode)

## Installation

<CodeGroup>
```bash npm
npm install @signum/sdk
```

```bash yarn
yarn add @signum/sdk
```

```bash bun
bun add @signum/sdk
```
</CodeGroup>

Or use direct HTTP requests with any language.

## Option A: Direct API (Embedded Mode)

For applications that handle authentication in their own UI:

<Steps>
  <Step title="Initialize Client">
```typescript
import { SignumClient } from '@signum/sdk';

const signum = new SignumClient({
  baseUrl: 'https://api.signum.id',
  // For development: 'http://localhost:5000'
});
```
  </Step>

  <Step title="Register or Login">
```typescript
// Register new user
const session = await signum.auth.register({
  email: 'user@example.com',
  password: 'secure-password-123',
  displayName: 'John Doe',
});

// Or login existing user
const session = await signum.auth.login({
  email: 'user@example.com',
  password: 'secure-password-123',
});

console.log('User:', session.user.email);
console.log('Token:', session.accessToken);
```
  </Step>

  <Step title="Access Protected Resources">
```typescript
// Get user profile with chain identities
const user = await signum.auth.me(session.accessToken);

console.log('Solana wallets:', user.chainIdentities?.solana);
console.log('EVM wallets:', user.chainIdentities?.evm);
console.log('KYC status:', user.kyc?.verified);
```
  </Step>

  <Step title="Create Wallets">
```typescript
// Create embedded wallets (EVM + Solana)
const wallets = await signum.accounts.createWallets(session.accessToken);

console.log('EVM:', wallets.evm.address);
console.log('Solana:', wallets.solana.address);

// Or create Solana PDA specifically
const pda = await signum.solana.createWallet(session.accessToken);
console.log('PDA:', pda.wallet.address);
```
  </Step>
</Steps>

## Option B: OAuth2 (Third-Party Integration)

For applications that want to use Signum as an identity provider:

<Steps>
  <Step title="Initialize OAuth Client">
```typescript
import { SignumOAuth } from '@signum/sdk';

const oauth = new SignumOAuth({
  clientId: 'your_client_id',
  redirectUri: 'https://yourapp.com/callback',
  scopes: ['openid', 'profile', 'email', 'chain:read'],
});
```
  </Step>

  <Step title="Redirect to Signum">
```typescript
// Generate auth URL with PKCE
const { url, codeVerifier, state } = oauth.getAuthorizationUrl();

// Store for callback validation
sessionStorage.setItem('code_verifier', codeVerifier);
sessionStorage.setItem('oauth_state', state);

// Redirect user
window.location.href = url;
```
  </Step>

  <Step title="Handle Callback">
```typescript
// On callback page (e.g., /callback)
const params = new URLSearchParams(window.location.search);
const code = params.get('code');

// Validate state
const returnedState = params.get('state');
if (returnedState !== sessionStorage.getItem('oauth_state')) {
  throw new Error('Invalid state');
}

// Exchange code for tokens
const codeVerifier = sessionStorage.getItem('code_verifier');
const tokens = await oauth.exchangeCode(code, codeVerifier);

console.log('Access Token:', tokens.accessToken);
console.log('ID Token:', tokens.idToken);
```
  </Step>

  <Step title="Get User Info">
```typescript
const userInfo = await oauth.getUserInfo(tokens.accessToken);

console.log('User ID:', userInfo.sub);
console.log('Email:', userInfo.email);
console.log('Chains:', userInfo['https://signum.id/claims/chain_identities']);
```
  </Step>
</Steps>

## Verify KYC Status

Check and initiate identity verification:

```typescript
// Check current status
const status = await signum.kyc.getStatus(accessToken);

if (status.verified) {
  console.log('KYC Level:', status.level);
  console.log('Verified:', status.verifiedAt);
} else if (status.status === 'not_started') {
  // Initiate verification
  const verification = await signum.kyc.initiate(accessToken, {
    level: 'enhanced',
    userInfo: {
      firstName: 'John',
      lastName: 'Doe',
      email: user.email,
    },
  });
  
  // Use verification.sdkUrl or verification.accessToken
  // with Sumsub WebSDK to display verification UI
  console.log('Start verification:', verification.sdkUrl);
}
```

## Development Mode

For local development without production credentials:

```bash
# Start the Signum app locally
cd signum-repos/app
bun run dev
# Server runs on http://localhost:5000
```

```typescript
const signum = new SignumClient({
  baseUrl: 'http://localhost:5000',
});

// Development features:
// - Mock KYC provider (auto-approve)
// - Local JWT signing
// - In-memory token storage
```
