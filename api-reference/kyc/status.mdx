# Get KYC Status

Retrieve the current KYC verification status for the authenticated user.

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token with `kyc:status` scope.
</ParamField>

## Response

<ResponseField name="verified" type="boolean">
  Whether KYC is verified and valid.
</ResponseField>

<ResponseField name="level" type="string">
  Verification level: `basic`, `enhanced`, or `institutional`.
</ResponseField>

<ResponseField name="status" type="string">
  Current status: `not_started`, `pending`, `in_review`, `approved`, `rejected`, `expired`.
</ResponseField>

<ResponseField name="verifiedAt" type="string">
  ISO 8601 timestamp of verification (if approved).
</ResponseField>

<ResponseField name="expiresAt" type="string">
  ISO 8601 timestamp when verification expires.
</ResponseField>

<ResponseField name="rejectionReason" type="string">
  Reason for rejection (if rejected).
</ResponseField>

<ResponseField name="canRetry" type="boolean">
  Whether the user can retry verification (if rejected).
</ResponseField>

<RequestExample>
```bash cURL
curl https://api.signum.id/kyc/status \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```typescript TypeScript
const response = await fetch('https://api.signum.id/kyc/status', {
  headers: { Authorization: `Bearer ${accessToken}` },
});

const status = await response.json();
if (status.verified) {
  console.log('KYC verified at level:', status.level);
} else if (status.status === 'pending') {
  console.log('Verification in progress');
}
```
</RequestExample>

<ResponseExample>
```json 200 OK (Verified)
{
  "verified": true,
  "level": "enhanced",
  "status": "approved",
  "verifiedAt": "2026-01-15T14:00:00Z",
  "expiresAt": "2027-01-15T14:00:00Z"
}
```

```json 200 OK (Pending)
{
  "verified": false,
  "level": null,
  "status": "in_review",
  "verifiedAt": null,
  "expiresAt": null
}
```

```json 200 OK (Rejected)
{
  "verified": false,
  "level": null,
  "status": "rejected",
  "rejectionReason": "Document expired",
  "canRetry": true
}
```
</ResponseExample>
