# Create Embedded Wallets

Create Privy-managed embedded wallets for EVM and Solana networks. These wallets are MPC-secured with no seed phrase.

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token from login or OAuth2 flow.
</ParamField>

## Response

<ResponseField name="evm" type="object">
  Created EVM wallet.
  <Expandable title="properties">
    <ResponseField name="address" type="string">
      EVM wallet address (same on all chains).
    </ResponseField>
    <ResponseField name="type" type="string">
      Always "embedded".
    </ResponseField>
    <ResponseField name="chains" type="number[]">
      Supported chain IDs.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="solana" type="object">
  Created Solana wallet.
  <Expandable title="properties">
    <ResponseField name="address" type="string">
      Solana wallet address.
    </ResponseField>
    <ResponseField name="type" type="string">
      Always "embedded".
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.signum.id/accounts/wallets \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```typescript TypeScript
const response = await fetch('https://api.signum.id/accounts/wallets', {
  method: 'POST',
  headers: { Authorization: `Bearer ${accessToken}` },
});

const wallets = await response.json();
console.log('EVM:', wallets.evm.address);
console.log('Solana:', wallets.solana.address);
```
</RequestExample>
