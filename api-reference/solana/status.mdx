# Get Solana Status

Check the authenticated user's Solana PDA wallet status and attestation information. This is a **query endpoint** that reads from the database.

<Note>
  This endpoint reads from the database, which is eventually consistent with on-chain state. The `status` field indicates the current synchronization state.
</Note>

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token from login or OAuth2 flow.
</ParamField>

## Response

<ResponseField name="exists" type="boolean">
  Whether a wallet record exists for the user.
</ResponseField>

<ResponseField name="address" type="string">
  Derived PDA wallet address.
</ResponseField>

<ResponseField name="status" type="string">
  Wallet status: "pending", "confirmed", "failed", or "unknown".
</ResponseField>

<ResponseField name="embeddedWallet" type="string">
  Associated embedded wallet address (from Privy).
</ResponseField>

<ResponseField name="attestationPda" type="string">
  Derived attestation PDA address.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of wallet creation.
</ResponseField>

### Status Values

| Status | Description |
|--------|-------------|
| `pending` | PDA creation has been queued but not yet confirmed on-chain |
| `confirmed` | PDA is initialized and confirmed on-chain |
| `failed` | PDA initialization failed after retries |
| `unknown` | No wallet record exists for this user |

<RequestExample>
```bash cURL
curl https://api.signum.id/solana/status \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```typescript TypeScript
const response = await fetch('https://api.signum.id/solana/status', {
  headers: { Authorization: `Bearer ${accessToken}` },
});

const status = await response.json();
if (status.status === 'confirmed') {
  console.log('Wallet ready:', status.address);
} else if (status.status === 'pending') {
  console.log('Wallet creation in progress...');
}
```
</RequestExample>
