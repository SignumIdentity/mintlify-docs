# Get Solana Status

Check if the authenticated user has a Solana PDA wallet and their attestation status.

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token from login or OAuth2 flow.
</ParamField>

## Response

<ResponseField name="hasWallet" type="boolean">
  Whether the user has a Solana PDA wallet.
</ResponseField>

<ResponseField name="wallet" type="object">
  Wallet details if exists.
  <Expandable title="properties">
    <ResponseField name="address" type="string">
      Solana wallet address.
    </ResponseField>
    <ResponseField name="type" type="string">
      Wallet type: "pda" or "linked".
    </ResponseField>
    <ResponseField name="createdAt" type="string">
      ISO 8601 timestamp.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="attestation" type="object">
  KYC attestation status on Solana.
  <Expandable title="properties">
    <ResponseField name="verified" type="boolean">
      Whether attestation is valid.
    </ResponseField>
    <ResponseField name="level" type="string">
      Verification level.
    </ResponseField>
    <ResponseField name="anchoredAt" type="string">
      When attestation was created on-chain.
    </ResponseField>
    <ResponseField name="expiresAt" type="string">
      When attestation expires.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl https://api.signum.id/solana/status \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```typescript TypeScript
const response = await fetch('https://api.signum.id/solana/status', {
  headers: { Authorization: `Bearer ${accessToken}` },
});

const status = await response.json();
if (status.hasWallet) {
  console.log('Wallet:', status.wallet.address);
  console.log('KYC Verified:', status.attestation?.verified);
}
```
</RequestExample>
