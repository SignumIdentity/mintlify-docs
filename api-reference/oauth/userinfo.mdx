# UserInfo

Retrieve user profile claims based on granted scopes. This is the standard OIDC UserInfo endpoint.

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer access token with `openid` scope.
</ParamField>

## Response

Claims returned depend on granted scopes:

| Scope | Claims |
|-------|--------|
| `openid` | `sub` |
| `profile` | `name`, `picture` |
| `email` | `email`, `email_verified` |
| `chain:read` | `https://signum.id/claims/chain_identities` |
| `kyc:status` | `https://signum.id/claims/kyc` |

<RequestExample>
```bash cURL
curl https://api.signum.id/oauth/userinfo \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```typescript TypeScript
const response = await fetch('https://api.signum.id/oauth/userinfo', {
  headers: { Authorization: `Bearer ${accessToken}` },
});

const userInfo = await response.json();
console.log(userInfo.email);
console.log(userInfo['https://signum.id/claims/chain_identities']);
```
</RequestExample>
