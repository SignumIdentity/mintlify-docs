# OAuth2 Authorize

Initiate the OAuth2 authorization code flow. Redirect users to this endpoint to authenticate.

## Query Parameters

<ParamField query="response_type" type="string" required>
  Must be `code` for authorization code flow.
</ParamField>

<ParamField query="client_id" type="string" required>
  Your application's client ID.
</ParamField>

<ParamField query="redirect_uri" type="string" required>
  URI to redirect after authorization. Must match registered URI.
</ParamField>

<ParamField query="scope" type="string" required>
  Space-separated list of scopes. See [Scopes](/authentication/scopes).
</ParamField>

<ParamField query="state" type="string">
  Random string for CSRF protection. Recommended.
</ParamField>

<ParamField query="code_challenge" type="string" required>
  PKCE code challenge (base64url-encoded SHA256 of code_verifier).
</ParamField>

<ParamField query="code_challenge_method" type="string" required>
  Must be `S256`.
</ParamField>

<ParamField query="nonce" type="string">
  Random string for ID token replay protection.
</ParamField>

## Response

On success, redirects to `redirect_uri` with:

| Parameter | Description |
|-----------|-------------|
| `code` | Authorization code (exchange for tokens) |
| `state` | Original state value |

On error, redirects with:

| Parameter | Description |
|-----------|-------------|
| `error` | Error code |
| `error_description` | Human-readable description |
| `state` | Original state value |
