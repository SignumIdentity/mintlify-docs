# JSON Web Key Set (JWKS)

Returns the public keys used to verify JWT signatures.

## Response

Standard JWKS format with ES256 public keys.

<RequestExample>
```bash cURL
curl https://api.signum.id/.well-known/jwks.json
```

```typescript TypeScript
import * as jose from 'jose';

const JWKS = jose.createRemoteJWKSet(
  new URL('https://api.signum.id/.well-known/jwks.json')
);

// Verify a token
const { payload } = await jose.jwtVerify(token, JWKS, {
  issuer: 'https://api.signum.id',
  audience: 'your_client_id',
});
```
</RequestExample>
