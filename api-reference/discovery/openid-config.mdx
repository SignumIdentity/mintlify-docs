# OpenID Configuration

Returns the OpenID Connect discovery document containing endpoints and capabilities.

## Response

Standard OIDC discovery document fields.

<RequestExample>
```bash cURL
curl https://api.signum.id/.well-known/openid-configuration
```
</RequestExample>
